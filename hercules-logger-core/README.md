# Logger core

## Описание 
Ядро для клиентских библиотек отправки метрик в Hercules Gateway.

## Параметры запуска
Встраивается в pom.xml
```xml
<dependency>
    <groupId>ru.kontur.vostok.hercules</groupId>
    <artifactId>hercules-logger-core</artifactId>
    <version>{version}</version>
</dependency>
```
